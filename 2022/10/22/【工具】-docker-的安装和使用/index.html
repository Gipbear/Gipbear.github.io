docker 安装

以 Windows 为例，首先进入 docker 官网，找到对应 docker-desktop 选择 Windows 版本进行下载。下载之后双击安装，稍等片刻就安装完成了。

有很多操作直接在 docker-desktop 上可以交互式操作，会比较简单，但是学习 docker 命令对后面还是有好处的，毕竟之后工作中有 docker 的时候并不一定会有相应的 desktop 工具。
